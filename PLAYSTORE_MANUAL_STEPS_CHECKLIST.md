# Play Store Manual Steps Checklist

These steps are manual in Google Play Console and cannot be completed automatically from this repository.

## Console setup
- [ ] Create/verify Play Console developer account
- [ ] Create app entry (App name, default language, app/game, free/paid)
- [ ] Enable Play App Signing

## Policy forms
- [ ] App content completed
- [ ] Data safety completed
- [ ] Content rating completed
- [ ] Ads declaration completed (if needed)
- [ ] Government app declaration completed (if needed)

## Store listing
- [ ] Upload app icon (512x512)
- [ ] Upload feature graphic (1024x500)
- [ ] Upload screenshots
- [ ] Paste short/full descriptions
- [ ] Set support contact details
- [ ] Set privacy policy URL

## Release flow
- [ ] Upload AAB to Internal testing
- [ ] Add release notes
- [ ] Add tester list
- [ ] Verify test feedback and crashes
- [ ] Promote to production with staged rollout
