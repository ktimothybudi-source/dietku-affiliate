# App Store Connect Metadata Checklist (v1.0)

## App Description updates

- [ ] Clearly mark premium-only features with text like: `Requires Premium subscription`.
- [ ] Add Terms of Use (EULA) link in description (if using Apple standard EULA).
- [ ] Keep wording aligned with actual in-app access (no implied free access for paid features).

Suggested disclosure line:

`Some features (including advanced AI scan and premium insights) require a DietKu Premium subscription.`

## Screenshots updates

- [ ] Add visual label on premium screenshots: `Premium Feature` or `Requires Premium`.
- [ ] Remove any screenshot copy that implies premium features are included for free.

## Required legal links

- [ ] Privacy Policy URL populated in App Store Connect Privacy Policy field.
- [ ] Terms of Use URL in App Description or custom EULA field.
